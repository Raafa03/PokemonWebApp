About Us

A Pokemon Web App representa o resultado de um projeto concebido no contexto da disciplina de Bases de Dados II, integrada no programa académico do curso TeSP - Tecnologias e Programação de Sistemas de Informação, ministrado pela Escola Superior de Gestão e Tecnologia do Instituto Politécnico de Santarém, em Vila Franca de Xira. Este projeto tem como base um conjunto de dados fornecido pela dedicada Professora Vânia Besouro, que leciona a referida disciplina.
